# Rock Paper Scissors Game (Python)

A simple Python console game where you play Rock, Paper, Scissors against the computer. Tracks your wins and the computerâ€™s wins in each session.

## How to Play

- Run `rock_paper_scissors.py` using Python 3.
- Type Rock, Paper, or Scissors as your move, or Q to quit.
- The computer picks randomly and announces its choice.
- The game tracks your wins and the computer's wins until you decide to quit.

## Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

## Features

- Easy-to-understand command-line game
- Input validation for user entries
- Live score tracking
